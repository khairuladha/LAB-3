//answer all following quastions //

1. What style would you use to change the font of a block quote to Times Roman or if Times Roman not available to change to serif font?

<p style="font-family: arial,helvetica,sans-serif;"></p>
2. What style would you use to set the size of paragraph text to 12 point type?
<p style="font-size: normal;"></p>
3. What style would you use to underline all your headings and display heading text in uppercase letter?
h1,h2,h3,h4,h5,h6{ background-color: silver; text-transform: uppercase; }
4. What style would you use to center your paragraph text and display it in boldface italics?
<p><b><i>This text is italic</i></b></p>
5. What style would you use to display your unordered lists with the list.gif background image centered in the background?

ul{ list-style: circle url(apple.jpg) outside }

6. What style would you use to display your ordered lists with an integer label?

<ol> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </ol>
